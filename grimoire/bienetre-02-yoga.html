1 Debout

1.1 Vrikshasana

Pour

  • Force de la colonne vertébrale
  • Équilibre
  • Coordination neuro-musculaire

Contre

  • Sciatique
  • Pied plat

Asana

  1. Debout, bien droit, sur un pied
  2. L’autre pied est posé sur l’intérieur de la cuisse opposée
  3. Mains vers le haut, les paumes jointes
  4. Même mouvement en inversant le rôle des pieds

2 Assis

2.1 Vajarasana

  1. Assis sur les genoux
  2. Les mains sur les cuisses

2.2 Padmasana

Ressemble à la position du lotus

Pour

  • Relaxation
  • Énergie
  • Digestion
  • Posture

Asana

  1. Le pied droit est posé, plante vers le haut, sur la cuisse gauche
  2. Le pied gauche est posé, plante vers le haut, sur la cuisse droite

2.3 Shashankasana

  • Assis sur les genoux
  • Le buste, la tête et les mains inclinés vers l’avant, jusqu’au sol

2.4 Ustrasana

Pour

  • Ouverture de la poitrine et du torse
  • Renforce le dos et les épaules

Contre

  • Règles douloureuses

Asana

  1. Assis sur les genoux
  2. Les mains sur les talons
  3. Enchaîner avec Shashankasana

2.5 Paschimottanasana

Pour

  • Reins
  • Foie
  • Utérus
  • Ovaires
  • Appétit
  • Calme du mental

Contre

  • Guérison
  • Obésité
  • Dépression
  • Stress

Asana

  1. Assis, les jambes étendues vers l’avant
  2. Le buste incliné vers l’avant
  3. Les doigts touchent les orteils

2.6 Ardha Matsyendrasana

Pour

  • Ouverture du buste
  • Augmente l’arrivée d’oxygène dans les poumons

Contre

  • Tension dans le dos
  • Infection urinaire
  • Problèmes de règles

Asana

  1. Assis, les jambes étendues vers l’avant
  2. Placer la jambe gauche vers la droite sous la jambe droite qui est elle fléchie et orientée vers la gauche
  3. Étendre le bras gauche à droite du genou droit, la main sur la cheville droite
  4. Étendre le bras droit derrière le dos, la main sur la hanche gauche

3 À quatre pattes

3.1 Marjariasana

Pour

  • Flexibilité et force de la colonne vertébrale
  • Circulation sanguine
  • Tonifie l’abdomen

Asana

  1. À quatre pattes, les mains et les genoux
  2. Alterner
    • Dos courbé vers le haut, tête vers le bas
    • Dos bien droit, tête vers le haut

4 Allongé sur le ventre

4.1 Bhujangasana

Pour

  • Régulation du métabolisme
  • Équilibre du poids

Contre

  • Sciatique
  • Asthme
  • Fatigue
  • Stress

Asana

  1. Allongé sur le ventre
  2. Les mains à plat sur le sol
  3. Pousser sur les bras en relevant la tête et la poitrine
  4. Idéalement, le visage doit regarder vers le haut

4.2 Dhanurasana

Pour

  • Stimule les organes de reproduction

Contre

  • Problèmes aux reins
  • Stress

Asana

  1. Allongé sur le ventre
  2. Relever les mollets
  3. Tenir les chevilles avec les mains
  4. Relever la tête et la poitrine en tirant sur les bras

5 Allongé sur le dos

5.1 Shavasana

Pour

  • Relaxation
  • Concentration
  • Mémoire

Contre

  • Anxiété
  • Tension sanguine élevée

Asana

  1. Allongé sur le dos
  2. Jambes légèrement écartées
  3. Mains légèrement écartées du corps
  4. Paumes vers le haut

5.2 Navasana

Pour

  • Tonifie les muscles des bras et des jambes
  • Stabilise
  • Augmente la capacité de concentration
  • Active la prostate
  • Active les reins
  • Active les intestins
  • Active la thyroïde

Asana

  1. Allongé sur le dos
  2. Soulever le buste et les jambes du sol
  3. Étendre les jambes en avant vers les genoux

5.3 Pawanmuktasana

Pour

  • Renforce les muscles abdominaux
  • Masse les intestins
  • Masse les organes internes du système digestif
  • Digestion

Contre

  • Tension dans le bas du dos
  • Graisse dans les cuisses, les fesses et l’abdomen

Asana

  1. Allongé sur le dos
  2. Replier les cuisses sur le ventre, et les mollets sur les cuisses
  3. Tenir les mollets avec les bras croisés
  4. Redresser la tête vers les genoux

5.4 Matsyasana

Pour

  • Absorption des nutriments
  • Bonne méthode de respiration
  • Étire la poitrine
  • Étire le cou

Contre

  • Problèmes respiratoires
  • Tension dans les épaules
  • Tension dans le cou

Asana

  1. Allongé sur le dos
  2. Mains sous les fesses
  3. Redresser légèrement le buste, plier légèrement les coudes
  4. Tendre le cou pour que le sommet de la tête reste posée au sol

6 Autre

6.1 Halasana

Pour

  • Régulation du taux de sucre dans le sang
  • Régulation du métabolisme

Contre

  • Diabète
  • Symptômes de la ménopause
  • Stimule le système de reproduction
  • Surpoids

Asana

  1. Allongé sur le dos
  2. Tourner les jambes de 270° vers la tête jusqu’à ce que les pieds touchent le sol
  3. Le dos se redresse naturellement dans le mouvement
  4. Placer les coudes au sol et les mains sur le dos pour aider le mouvement
  5. Tendre les bras au sol et croiser les mains

6.2 Sarvangasana

Pour

  • Calme le cerveau
  • Digestion
  • Régulation du métabolisme

Contre

  • Dépression
  • Stress
  • Sinusite
  • Asthme
  • Stérilité

Asana

  1. Allongé sur le dos
  2. Lever les jambes vers le ciel
  3. Tourner encore les jambes de 90° vers la tête pour aider le dos à se redresser
  4. Dès que le dos est bien droit, placer les coudes au sol et les mains sur le dos pour le soutenir
  5. Redresser les jambes, pieds vers le haut
