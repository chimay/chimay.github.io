** Socle

  - Pi

    + π = périmètre cercle / diamètre

  - Nombre d’or

    + Φ - 1 =  1 / Φ

    + Φ = (1 + √5) / 2

    + Φ^2 = Φ + 1

    + (Φ - 1)^2 = 2 - Φ

    + 1 - 1 / Φ = 1 / Φ^2

    + 2 - Φ = 1 / Φ^2

  - Nombre d’argent

    + Δ - 2 = 1 / Δ

    + Δ = 1 + √2

    + Δ x (Δ - 2) = (√2 + 1) x (√2 - 1) = (√2)^2 - 1 = 2 - 1 = 1

  - Nombre de bronze

    + β - 3 = 1 / β

    + β = (3 + √13) / 2

  - Relations approximatives

    + π = ~ 22 / 7

    + Φ = ~ 1.618

    + Δ = ~ 2.414

    + β = ~ 3.303
