  - pi

  - nombre d’or
    + Φ - 1 =  1 / Φ
    + Φ = (1 + √5) / 2
    + Φ^2 = Φ + 1
    + (Φ - 1)^2 = 2 - Φ
    + 1 - 1 / Φ = 1 / Φ^2

  - nombre d’argent
    + Δ - 2 = 1 / Δ
    + Δ = 1 + √2
    + Δ x (Δ - 2) = (√2 + 1) x (√2 - 1) = (√2)^2 - 1 = 2 - 1 = 1
  - nombre de bronze
