* Éléments fondamentaux d’une histoire

  - décors
  - personnages
    + principaux
      * héros : sympathiques
      * opposants : antipathiques
    + secondaires
      * assistants des personnages principaux


* Phases de l’action

  - introduction
    + présentation des personnages
    + description des décors
  - problèmes qui démarrent l’action
    + suite d’aventures
    + demi-réussites
    + nouveaux problèmes
  - climax
    + on ne voit pas comment les héros vont s’en sortir
  - solution inattendue
  - final


* Présentation des personnages

  - souvent indirecte
  - on passe par
    + un autre personnage qui rencontre celui qu’on veut présenter
