2.10.1. Taoisme

2.10.1.1. Trigrammes

Un trigramme est l’association de trois symboles, chacun d’entre-eux pouvant représenter le yin ou le yang.

Le yang est généralement représenté par un trait plein et le yin par un trait redoublé. Nous les notons :

¦ Yang
— Yin

Voici les 8 possibilités des trigrammes :

¦¦¦ Ciel
¦¦— Lac, Brume, Marais
¦—¦ Feu
¦—— Foudre, Tonnerre
—¦¦ Vent, Bois
—¦— Eau
——¦ Montagne
——— Terre

Sous la forme des dualités, il donne :

Trigrammes Symboles Qualités
¦¦¦ Ciel Créativité, Force, Initiative
——— Terre Réceptivité, Adaptabilité, Accueil
¦¦— Lac, Brume, Marais Joie, Légèreté
——¦ Montagne Immobilité, Rigueur, Cohésion, Calme, Solidité
¦—¦ Feu Clarté, Lucidité, Vivacité, Éclat
—¦— Eau Profondeur, Endurance, Peur
¦—— Foudre, Tonnerre Dureté, Impulsion, Secousse
—¦¦ Vent, Bois Douceur, Pénétration, Soumission, Intériorisation
2.10.1.2. Hexagramme

Un hexagramme un constitué de deux trigrammes.

2.10.1.3. Bāguà

Les trigrammes sont représentés habituellement sous forme d’octogone entourant le tai-ji, l’ensemble étant alors nommé bāguà, ou plus simplement bagua.

4.4. Chine

4.4.1. Éléments du wu xing

xing = forme, mouvement

  • mu : bois
  • huo : feu
  • tu : terre
  • jin : métal
  • shul : eau

4.4.2. Cycles

  • sheng : cycle nourissier
    • le bois nourrit le feu
    • le feu nourrit la terre
    • la terre nourrit le métal
    • le métal nourrit l’eau
    • l’eau nourrit le bois
  • ke : cycle de contrôle
    • la bois contrôle la terre
    • la terre contrôle l’eau
    • l’eau contrôle le feu
    • le feu contrôle le métal
    • le métal contrôle le bois
  • xie : cycle d’affaiblissement, manque de sheng
    • le bois affaiblit l’eau
    • l’eau affaiblit le métal
    • le métal affaiblit la terre
    • la terre affaiblit le feu
    • le feu affaiblit le bois
  • cheng : cycle d’appauvrissement, excès de ke
    • le bois appauvrit la terre
    • la terre appauvrit l’eau
    • l’eau appauvrit le feu
    • le feu appauvrit le métal
    • le métal appauvrit le bois
  • wu : cycle de détérioration, manque de ke
    • le bois détériore le métal
    • le métal détériore le feu
    • le feu détériore l’eau
    • l’eau détériore la terre
    • la terre détériore le bois
