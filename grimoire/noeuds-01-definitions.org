* Introduction

Ce tome contient une série de définitions élémentaires ainsi qu’un
catalogue de structures essentielles que l’on retrouve en tant que
composants ou étapes de construction dans de nombreux noeuds.

* Orientation

  - avant :: côté du noeud situé vers vous

    Imaginez une table remplie d’objets : il y a ceux qui sont devant
    vous, à l’avant, tandis que les autres sont placés derrière les
    premiers, à l’arrière.

    L’avant est aussi la partie frontale du noeud, si vous imaginez
    qu’il vous regarde.

  - arrière :: côté du noeud opposé à l’avant

    L’arrière est donc derrière l’avant.

  - gauche et droite :: sens usuels

  - haut et bas :: sens usuels

  - poste :: objet auquel on attache la corde

    Peut être une barre, un anneau, ou une autre corde

  - avant du poste :: côté du poste situé devant celui-ci

  - arrière du poste :: côté du poste situé derrière celui-ci

    L’arrière du poste est opposé à l’avant.

  - directions par défaut :: pour les noeuds d’attache autour d’un
    poste horizontal, les directions par défaut changent :

    + arrière devient haut
    + avant devient bas
    + haut devient avant
    + bas devient arrière

    Comme si on prenait le noeud disposé sur une table pour le placer
    verticalement.

* Corde

  - partie libre :: partie de la corde qui mène à l’une de ses
    extrémités sans rencontrer de croisement ou de noeud

  - terminus :: tronçon de corde qui contient :

    + une des extrémités de la corde
    + la partie de la corde proche de cette extrémité

    La longueur d’un terminus s’étend en général sur quelques
    centimètres.

    Une corde possède donc deux terminus : un par extrémité.

  - partie de travail :: partie libre active de la corde, celle qui se
    déplace pour réaliser le noeud

    /Par défaut/ : lorsqu’on réalise une opération, c’est avec la
    partie de travail

  - courant :: synonyme de partie de travail

  - partie statique :: partie libre passive de la corde

  - dormant :: synonyme de partie statique

  - partie réglable :: partie de la corde permettant d’ajuster la
    taille de l’anse ou de la boucle d’un noeud en tirant

** Plusieurs cordes

  - partie statique d’un terminus :: raccourci qui signifie : partie
    statique de la corde sur laquelle est située le terminus

  - partie statique d’une partie de travail :: raccourci qui
    signifie : partie statique de la corde sur laquelle est située la
    partie de travail

** Cordes à torons

  - corde à toron :: corde complexe dont les éléments sont assemblés
    en mouvement hélicoïdal par torsion

  - toron :: élément d’une corde complexe à torons

    Il peut s’agir :

    + d’une corde simple
    + d’une corde complexe
    + d’une partie libre de corde

  - corde toronnée :: synonyme de corde à torons

  - toronner :: assembler une corde plus épaisse à partir de cordes
    plus fines, en les faisant tourner les unes autour des autres en
    un mouvement hélicoïdal

* Cercle

  - faire un cercle horlogique :: faire un cercle en tournant dans le
    sens horlogique

  - faire un cercle anti-horlogique :: faire un cercle en tournant
    dans le sens anti-horlogique

  - faire un demi-cercle horlogique :: faire un demi-cercle en
    tournant dans le sens horlogique

  - faire un demi-cercle anti-horlogique :: faire un demi-cercle en
    tournant dans le sens anti-horlogique

  - faire un tour horlogique :: faire un tour en tournant dans le sens
    horlogique

  - faire un tour anti-horlogique :: faire un tour en tournant dans le
    sens anti-horlogique

  - faire un demi-tour horlogique :: faire un demi-tour en
    tournant dans le sens horlogique

  - faire un demi-tour anti-horlogique :: faire un demi-tour en
    tournant dans le sens anti-horlogique

  - faire un quart de tour horlogique :: faire un quart de tour en
    tournant dans le sens horlogique

  - faire un quart de tour anti-horlogique :: faire un quart de tour
    en tournant dans le sens anti-horlogique

* Anse

  - faire une anse :: faire demi-tour en dessinant une forme arrondie,
    revenir près de la partie statique, et la longer sur une certaine
    distance

    Ressemble vaguement à un champignon.

  - faire une anse, méthode du pliage :: plier la corde à l’endroit
    voulu pour lui faire prendre la forme d’une anse

    C’est la méthode la plus utilisée : elle est rapide, et ne
    nécessite pas d’avoir accès à l’extrémité de la corde.

  - voûte d’une anse :: partie de l’anse où la corde fait demi-tour en
    dessinant une forme arrondie

    /Par défaut/ : la voûte de l’anse est à l’arrière

  - extrémité d’une anse :: synonyme de voûte de l’anse

  - base d’une anse :: partie de l’anse opposée à sa base

    La base est aussi la frontière entre l’anse et le reste de la
    corde.

    /Par défaut/ : la base de l’anse est à l’avant

  - bord d’une anse :: partie de corde de l’anse située entre sa base
    et sa voûte

  - faire une anse horlogique :: faire une anse en tournant d’un
    demi-tour dans le sens horlogique

  - faire une anse anti-horlogique :: faire une anse en tournant d’un
    demi-tour dans le sens anti-horlogique

  - faire une anse, à droite :: faire une anse, partie de travail à
    droite

    /Par défaut/ : la voûte est en arrière et cette anse est donc
    horlogique.

  - faire une anse, à gauche :: faire une anse, partie de travail à
    gauche

    /Par défaut/ : la voûte est en arrière et cette anse est donc
    anti-horlogique.

  - faire une ganse :: dessiner une forme de cercle ouvert

    La ganse est en fait une anse large.

    Anse & ganse sont interchangeables, il suffit d’écarter un peu
    plus les bords d’une anse pour former une ganse.

** Passage

  - passer dans une anse :: passer au travers d’une anse

  - entrer dans une anse par au-dessus :: passer au travers d’une anse
    en entrant par au-dessus

    Une entrée implique une sortie, qui se fait forcément sous l’anse.

    L’endroit de la sortie peut être précisé, si nécessaire : bord
    gauche ou droit de l’anse, voûte, base, etc.

  - entrer dans une anse par en-dessous :: passer au travers d’une
    anse en entrant par en-dessous

    La sortie se fait forcément au-dessus de l’anse.

  - passer une anse *A* dans une anse *B* :: passer *A* au travers de *B*

** Méta-anse

  - faire une méta-anse à partir d’une anse :: faire une anse
    ordinaire *A*, puis faire une anse double *M* à partir de la
    double corde formée par les bords de *A*

* Partie double

  - partie double :: partie complexe constituée de deux parties
    simples de corde

  - faire une doublure :: faire une longue anse et placer les bords
    l’un contre l’autre

    Une doublure peut être utilisée comme partie de travail.

  - faire un terminus double :: placer les deux terminus l’un contre
    l’autre

    Un terminus double peut être utilisé comme partie de travail.

* Boucle

  - faire une boucle :: faire un tour complet en dessinant une forme
    arrondie, puis croiser la partie statique

    Ressemble à une lettre E minuscule.

  - spire :: synonyme de boucle

  - faire une boucle, méthode par pliage et torsion :: plier la corde
    à l’endroit voulu pour lui faire prendre la forme d’une anse, puis
    faire tourner l’anse d’un demi-tour autour de l’axe reliant la
    base à la voûte pour créer une boucle

    Le sens de rotation détermine si la partie de travail se retrouve
    au-dessus ou en-dessous de la partie statique.

    C’est la méthode la plus utilisée : elle est rapide, et ne
    nécessite pas d’avoir accès à l’extrémité de la corde.

  - croisement d’une boucle :: endroit où les deux parties de la corde
    se croisent pour former la boucle

    Le croisement est aussi la frontière entre la boucle et le reste
    de la corde.

    /Par défaut/ : le croisement de la boucle est à l’avant

  - base d’une boucle :: synonyme de croisement de la boucle

  - voûte d’une boucle :: partie de la boucle opposée à son croisement

    /Par défaut/ : la voûte de la boucle est à l’arrière

  - extrémité d’une boucle :: synonyme de voûte de la boucle

  - bord d’une boucle :: partie de corde de la boucle située entre son
    croisement et sa voûte

  - faire une boucle horlogique :: faire une boucle en tournant dans
    le sens horlogique, puis en effectuant le croisement

  - faire une boucle anti-horlogique :: faire une boucle en tournant
    dans le sens anti-horlogique, puis en effectuant le croisement

  - faire une boucle, au-dessus :: faire un cercle, puis passer la
    partie de travail au-dessus de la partie statique pour effectuer
    le croisement

    En cas d’ambiguité, on précise : faire une boucle, partie de
    travail au-dessus.

  - faire une boucle, en-dessous :: faire un cercle, puis passer la
    partie de travail en-dessous de la partie statique pour effectuer
    le croisement

    En cas d’ambiguité, on précise : faire une boucle, partie de
    travail en-dessous.

  - faire une boucle, au-dessus vers *D* :: faire un cercle, puis
    passer la partie de travail au-dessus de la partie statique pour
    effectuer le croisement, de telle sorte que le croisement donne
    une direction *D* à la partie de travail

    En cas d’ambiguité, on précise : faire une boucle, partie de
    travail au-dessus vers *D*.

    On peut demander n’importe quelle direction pour la partie de
    travail : droite, gauche, avant, arrière, haut, bas, avant droit,
    arrière gauche, etc.

    /Par défaut// : si la partie de travail part d’un côté, la partie
    statique est orientée approximativement du côté opposé.

    /Exemple/ :

    + la partie de travail part principalement vers la droite,
      légèrement vers l’avant
    + la partie statique est orientée principalement vers la gauche,
      légèrement vers l’avant.

  - faire une boucle, en-dessous vers *D* :: faire un cercle, puis
    passer la partie de travail en-dessous de la partie statique pour
    effectuer le croisement, de telle sorte que le croisement donne
    une direction *D* à la partie de travail

    En cas d’ambiguité, on précise : faire une boucle, partie de
    travail en-dessous vers *D*.

  - faire une boucle, au-dessus vers la droite :: faire un cercle
    anti-horlogique, puis passer la partie de travail au-dessus de la
    partie statique pour effectuer le croisement, de telle sorte que
    la partie de travail parte vers la droite

    En pratique, la partie de travail part principalement vers la
    droite, légèrement vers l’avant, et la partie statique est
    orientée principalement vers la gauche, légèrement vers l’avant.

    /Par défaut/ : la voûte est en arrière et cette boucle est donc
    anti-horlogique.

  - faire une boucle supérieure droite :: synonyme de boucle
    au-dessus, vers la droite

  - faire une boucle, en-dessous vers la droite :: faire un cercle
    anti-horlogique, puis passer la partie de travail en-dessous de la
    partie statique pour effectuer le croisement, de telle sorte que
    la partie de travail parte vers la droite

    /Par défaut/ : la voûte est en arrière et cette boucle est donc
    anti-horlogique.

  - faire une boucle inférieure droite :: synonyme de boucle
    en-dessous, vers la droite

  - faire une boucle, au-dessus vers la gauche :: faire un cercle
    horlogique, puis passer la partie de travail au-dessus de la
    partie statique pour effectuer le croisement, de telle sorte que
    la partie de travail parte vers la gauche

    En pratique, la partie de travail part principalement vers la
    gauche, légèrement vers l’avant, et la partie statique est
    orientée principalement vers la droite, légèrement vers l’avant.

    /Par défaut/ : la voûte est en arrière et cette boucle est donc
    horlogique.

  - faire une boucle supérieure gauche :: synonyme de boucle
    au-dessus, vers la gauche

  - faire une boucle, en-dessous vers la gauche :: faire un cercle
    horlogique, puis passer la partie de travail en-dessous de la
    partie statique pour effectuer le croisement, de telle sorte que
    la partie de travail parte vers la gauche

    /Par défaut/ : la voûte est en arrière et cette boucle est donc
    horlogique.

  - faire une boucle inférieure gauche :: synonyme de boucle
    en-dessous, vers la gauche

** Torsion

  - tordre une anse :: faire tourner une anse d’un demi-tour autour de
    l’axe reliant la base à la voûte pour créer une boucle

  - faire une boucle en partant d’une anse :: synonyme de tordre une
    anse

  - tordre une boucle :: faire tourner la boucle autour de l’axe
    reliant sa base à sa voûte

    /Remarque/ :

    + si on se trompe de sens de rotation, la boucle se defait
    + une torsion d’un demi-tour transforme le croisement simple de la
      boucle en croisement double
    + une torsion de plusieurs demi-tours produit une boucle torsadée

  - torsader une boucle :: tordre une boucle de plusieurs demi-tours,
    ce qui produit une double hélice au niveau du croisement

** Passage

  - passer dans une boucle :: passer au travers d’une boucle

  - entrer dans une boucle par au-dessus :: passer au travers d’une
    boucle en entrant par au-dessus

    Une entrée implique une sortie, qui se fait forcément sous la boucle.

    L’endroit de la sortie peut être précisé, si nécessaire : bord
    gauche ou droit de la boucle, voûte, croisement, etc.

  - entrer dans une boucle par en-dessous :: passer au travers d’une
    boucle en entrant par en-dessous

    La sortie se fait forcément au-dessus de la boucle.

  - passer une anse *A* dans une boucle *B* :: passer *A* au travers de *B*

  - passer une boucle *B* dans une anse *A* :: passer *B* au travers de *A*

  - passer une boucle *B* dans une boucle *C* :: passer *B* au travers de *C*

* Bobine

  - faire une bobine :: superposer plusieurs cercles en effectuant
    plusieurs tours dans le même sens autour d’un axe de rotation
    commun

    /Remarques/ :

    + la forme obtenue est cylindrique
    + le premier cercle forme une boucle

  - faire une bobine de /N/ tours :: superposer /N/ cercles en
    effectuant /N/ tours dans le même sens autour d’un axe de rotation
    commun

  - faire une bobine, en progressant vers *D* :: faire une bobine en
    s’arrangeant pour que les centres des cercles progressent sur
    l’axe de rotation commun dans la direction *D*

* Lentille

  - lentille :: région située entre deux croisements de corde

  - bord gauche d’une lentille :: partie de corde délimitant la gauche
    de la lentille, entre les deux croisements

  - bord droit d’une lentille :: partie de corde délimitant la droite
    de la lentille, entre les deux croisements

* Esse

  - faire un esse :: faire une première anse, puis une seconde anse
    qui tourne dans l’autre sens et partage un bord avec la première
    anse

    /Exemple/ :

    1. faire une anse horlogique *H*
       * voûte à l’arrière (par défaut)
       * partie de travail à droite, vers l’avant
    2. faire un demi-tour anti-horlogique vers l’arrière, ce qui forme
       une anse *A* telle que
       * la voûte de *A* est à l’avant
       * la partie de travail est à droite, vers l’arrière
       * le bord gauche de *A* est aussi le bord droit de *H*

    Ressemble vaguement à une lettre S, ou à son reflet dans le
    miroir, d’où son nom.

  - faire un esse, à *C*, vers *D* :: faire un esse de telle sorte que
    la partie de travail termine la figure du côté *C* et en étant
    orientée dans la direction *D*

  - faire un esse, à l’avant, vers la droite :: faire un esse de telle
    sorte que la partie de travail termine la figure en étant orientée
    vers la droite

    /Détail/ :

    1. faire une anse horlogique *H*
       * voûte à droite
       * partie de travail à l’avant, vers la gauche
    2. faire un demi-tour anti-horlogique vers la droite, ce qui forme
       une anse *A* telle que
       * la voûte de *A* est à gauche
       * la partie de travail est à l’avant, vers la droite
       * le bord avant de *H* est aussi le bord arrière de *A*

  - faire un esse, à l’avant, vers la gauche :: faire un esse de telle
    sorte que la partie de travail termine la figure en étant orientée
    vers la gauche

    /Détail/ :

    1. faire une anse anti-horlogique *A*
       * voûte à gauche
       * partie de travail à l’avant, vers la droite
    2. faire un demi-tour horlogique vers la gauche, ce qui forme une
       anse *H* telle que
       * la voûte de *H* est à droite
       * la partie de travail est à l’avant, vers la gauche
       * le bord avant de *A* est aussi le bord arrière de *H*

    C’est l’orientation qui correspond à la lettre S.

  - faire un esse, à droite, vers l’avant :: faire un esse de telle
    sorte que la partie de travail termine la figure en étant à droite
    et orientée vers l’avant

    /Détail/ :

    1. faire une anse anti-horlogique *A*
       * voûte à l’avant
       * partie de travail à droite, vers l’arrière
    2. faire un demi-tour horlogique vers l’avant, ce qui forme une
       anse *H* telle que
       * la voûte de *H* est à l’arrière
       * la partie de travail est à droite, vers l’avant
       * le bord droit de *A* est aussi le bord gauche de *H*

  - faire un esse, à droite, vers l’arrière :: faire un esse de telle
    sorte que la partie de travail termine la figure en étant à droite
    et orientée vers l’arrière

    /Détail/ :

    1. faire une anse horlogique *H*
       * voûte à l’arrière (par défaut)
       * partie de travail à droite, vers l’avant
    2. faire un demi-tour anti-horlogique vers l’arrière, ce qui forme
       une anse *A* telle que
       * la voûte de *A* est à l’avant
       * la partie de travail est à droite, vers l’arrière
       * le bord droit de *H* est aussi le bord gauche de *A*

* Effe

  - faire un effe :: faire une première boucle, puis une seconde
    boucle qui partage le même croisement que la première, mais dont
    la voûte est dans la direction opposée. Les croisements doivent
    alterner les côtés dessus et dessous. Les sens de rotation des
    deux boucles sont opposés. Les directions de sortie des deux
    boucles sont les mêmes.

    Ressemble vaguement à une lettre F minuscule, ou à son reflet dans
    le miroir, d’où son nom.

    /Exemple/ :

    1. faire une boucle anti-horlogique *A*, en-dessous vers la droite
       * soit *X* le croisement de *A*
    2. faire une boucle horlogique *H*, dont le croisement s’effectue
       en passant au-dessus de *X*, vers la droite

    /Différence avec le esse/ : le esse est constitué de deux anses
    qui partagent un bord, tandis que le effe est constitué de deux
    boucles qui partagent un croisement.

  - boucles d’un effe :: un effe contient deux boucles

  - croisement d’un effe :: c’est le double croisement commun aux deux
    boucles du effe

  - faire un effe supérieur :: faire un effe de telle sorte que le
    croisement de la seconde boucle se fasse au-dessus du croisement
    de la première

  - faire un effe inférieur :: faire un effe de telle sorte que le
    croisement de la seconde boucle se fasse en-dessous du croisement
    de la première

  - faire un effe supérieur droit :: faire un effe supérieur, de telle
    sorte que la partie de travail parte vers la droite après le
    croisement de la seconde boucle

    /Détail/ :

    1. faire une boucle anti-horlogique *A*, en-dessous vers la droite
       * soit *X* le croisement de *A*
    2. faire une boucle horlogique *H*, dont le croisement s’effectue
       en passant au-dessus de *X*, vers la droite

  - faire un effe inférieur droit :: faire un effe inférieur, de telle
    sorte que la partie de travail parte vers la droite après le
    croisement de la seconde boucle

    /Détail/ :

    1. faire une boucle anti-horlogique *A*, au-dessus vers la droite
       * soit *X* le croisement de *A*
    2. faire une boucle horlogique *H*, dont le croisement s’effectue
       en passant sous *X*, vers la droite

  - faire un effe supérieur gauche :: faire un effe supérieur, de
    telle sorte que la partie de travail parte vers la gauche après le
    croisement de la seconde boucle

    /Détail/ :

    1. faire une boucle horlogique *H*, en-dessous vers la gauche
       * soit *X* le croisement de *H*
    2. faire une boucle anti-horlogique *A*, dont le croisement
       s’effectue en passant au-dessus de *X*, vers la gauche

  - faire un effe inférieur gauche :: faire un effe inférieur, de
    telle sorte que la partie de travail parte vers la gauche après le
    croisement de la seconde boucle

    /Détail/ :

    1. faire une boucle horlogique *H*, au-dessus vers la gauche
       * soit *X* le croisement de *H*
    2. faire une boucle anti-horlogique *A*, dont le croisement
       s’effectue en passant sous *X*, vers la gauche

* Huitain

  - faire un huitain :: faire une boucle, puis faire un demi-cercle en
    tournant dans l’autre sens, et croiser la partie statique un peu
    plus loin, de l’autre côté de la corde et dans la direction
    opposée

    Ressemble à un huit, d’où son nom. J’ai choisi le terme *huitain*
    pour ne pas confondre avec les noeuds en 8 (figure 8), mais cela
    n’a bien entendu rien à voir avec les strophes du même nom.

    Pour rendre l’opération plus facile, s’arranger pour que les
    parties libres fassent un angle droit après le croisement de la
    boucle.

    /Exemple/ :

    1. orienter la partie statique vers l’avant
    2. faire un cercle horlogique
    3. croiser la partie statique en-dessous vers la gauche
    4. faire un demi-cercle anti-horlogique
    5. croiser la partie statique au-dessus vers la droite, un peu plus
       en avant

  - faire un huitain en tordant une anse :: faire une anse, puis faire
    tourner l’anse d’un tour complet autour de l’axe reliant la base à
    la voûte pour qu’elle prenne la forme d’un huit.

    Il peut être nécessaire d’arranger les cordes pour que le huit
    apparaisse clairement.

    C’est la méthode la plus utilisée : elle est rapide, et ne
    nécessite pas d’avoir accès à l’extrémité de la corde.

  - faire un huitain en tordant une boucle :: faire une boucle, puis
    faire tourner la boucle d’un demi-tour autour de l’exe reliant le
    croisement à la voûte pour qu’elle prenne la forme d’un huit

    Le sens de rotation est le même que celui qui permet de passer
    d’une anse à la boucle.

    /Remarque/ : pour voir si la torsion s’est effectuée dans le bon
    sens, vérifier si, en partant de la partie statique, la corde
    passe alternativement en-dessous et au-dessus à chaque croisement
    rencontré

* Octave

  - faire une octave :: faire une boucle, puis faire une anse en
    tournant dans l’autre sens, et croiser la partie statique au même
    endroit, mais de l’autre côté de la corde et dans la direction
    opposée

    Ressemble à un huit, d’où son nom. J’ai choisi le terme *octave*
    pour ne pas confondre avec les noeuds en 8 (figure 8), mais cela
    n’a bien entendu rien à voir avec l’intervalle musical du même
    nom.

    Pour rendre l’opération plus facile, s’arranger pour que les
    parties libres fassent un angle droit après le croisement de la
    boucle.

    /Exemple/ :

    1. orienter la partie statique vers l’avant
    2. faire un cercle horlogique
    3. croiser la partie statique en-dessous, vers la gauche
    4. faire une anse anti-horlogique
    5. passer au-dessus du premier croisement, vers la droite

    /Différence avec le huitain/ : dans une octave, les deux
    croisements se font au même endroit.

  - boucle d’une octave :: c’est la boucle réalisée pour former le
    début de l’octave

  - anse d’une octave :: c’est l’anse réalisée pour former la suite du
    octave

    Autrement dit, c’est ce qui reste de l’octave lorsqu’on fait
    abstraction de sa boucle.

    L’octave est donc composé :

    + d’une boucle
    + d’une anse

  - croisement d’une octave :: c’est le double croisement commun à la
    boucle et à l’anse de l’octave

  - faire une octave supérieure :: faire une octave de telle sorte que
    le second croisement se fasse au-dessus du premier

  - faire une octave inférieure :: faire une octave de telle sorte que
    le second croisement se fasse en-dessous du premier

  - faire une octave supérieure droite :: faire une octave supérieure,
    de telle sorte que la partie de travail parte vers la droite après
    le second croisement

    /Détail/ :

    1. faire une boucle horlogique *A*, en-dessous vers la gauche
       * soit *X* le croisement de *A*
    2. faire une anse anti-horlogique *B*, puis passer au-dessus de
       *X*, vers la droite

  - faire une octave inférieure droite :: faire une octave inférieure,
    de telle sorte que la partie de travail parte vers la droite après
    le second croisement

    /Détail/ :

    1. faire une boucle horlogique *A*, au-dessus vers la gauche
       * soit *X* le croisement de *A*
    2. faire une anse anti-horlogique *B* et passer sous *X*, vers la
       droite

  - faire une octave supérieure gauche :: faire une octave supérieure,
    de telle sorte que la partie de travail parte vers la gauche après
    le second croisement

    /Détail/ :

    1. faire une boucle anti-horlogique *A*, en-dessous vers la droite
       * soit *X* le croisement de *A*
    2. faire une anse horlogique *B* et passer au-dessus de *X*, vers
       la gauche

  - faire une octave inférieure gauche :: faire une octave inférieure,
    de telle sorte que la partie de travail parte vers la gauche après
    le second croisement

    /Détail/ :

    1. faire une boucle anti-horlogique *A*, au-dessus vers la droite
       * soit *X* le croisement de *A*
    2. faire une anse horlogique *B* et passer sous *X*, vers la gauche

* Binocle

  - faire un binocle :: faire une anse, puis faire tourner l’anse d’un
    demi-tour autour de l’axe qui relie les deux bords, de façon à
    englober la base et une petite portion des parties libres de la
    corde

    Précisions :

    + ce sont les 2/3 de l’anse côté voûte qui tournent, et non
      l’entièreté de l’anse
    + puisque l’axe de rotation relie les bords, il est
      perpendiculaire à l’axe qui relie la base à la voûte

    Ressemble à une paire de lunettes, d’où son nom.

  - boucles d’un binocle :: un binocle contient deux boucles
    symétriques

  - bords intérieurs d’un binocle :: bords intérieurs des boucles, au
    centre du binocle

  - bords extérieurs d’un binocle :: bords extérieurs des boucles

    Si la voûte de l’anse est en avant, les bords extérieurs sont à
    gauche et à droite du binocle.

  - faire un binocle supérieur :: faire une anse, puis la faire
    tourner d’un demi-tour :

    + autour de l’axe qui relie les deux bords
    + en commençant à tourner vers le haut et l’avant

  - faire un binocle inférieur :: faire une anse, puis la faire
    tourner d’un demi-tour :

    + autour de l’axe qui relie les deux bords
    + en commençant à tourner vers le bas et l’avant

  - faire un binocle supérieur, méthode des deux boucles :: faire deux
    boucles comme suit :

    + faire une première boucle, au-dessus vers la droite, partie de
      travail venant de l’avant
    + faire une seconde boucle, juste à droite de la première, partie
      de travail en-dessous vers l’avant, partie statique venant de la
      gauche

  - faire un binocle inférieur, méthode des deux boucles :: faire deux
    boucles comme suit :

    + faire une première boucle, en-dessous vers la droite, partie de

    + faire une seconde boucle, juste à droite de la première, partie

* Bretzel

  - faire un bretzel :: faire une boucle, puis faire tourner la boucle
    d’un demi-tour autour de l’axe qui relie les deux bords, de façon
    à englober le croisement de la boucle

    Précisions :

    + ce sont les 2/3 de la boucle côté voûte qui tournent, et non
      l’entièreté de la boucle
    + puisque l’axe de rotation relie les bords, il est
      perpendiculaire à l’axe qui relie le croisement à la voûte

    Produit une forme comprenant deux boucles qui s’intersectent, avec
    une sorte de triangle au milieu. Ressemble à un huit couché et
    souligné, ou à un bretzel, d’où son nom.

  - boucles d’un bretzel :: un bretzel contient deux boucles qui
    s’intersectent

    Si on imagine un bretzel avec les voûtes à l’arrière, on a :

    + une première boucle avec la voûte en arrière à gauche et le
      croisement en avant à droite
    + une seconde boucle avec la voûte en arrière à droite et le
      croisement en avant à gauche

  - croisements d’un bretzel :: un bretzel contient trois
    croisements : un croisement par boucle, plus un croisement entre
    les bords des deux boucles

    Si on imagine un bretzel avec les voûtes en arrière, le troisième
    croisement s’effectue entre :

    + le bord droit de la boucle dont la voûte est en arrière à
      gauche
    + le bord gauche de la boucle dont la voûte est en arrière à
      droite

  - triangle d’un bretzel :: triangle dont les côtés relient les trois
    croisements du bretzel

    Le triangle est aussi l’intersection des deux boucles du bretzel.

    Les côtés du triangle peuvent être arrondis.

  - oreille gauche d’un bretzel :: partie de la boucle gauche du
    bretzel qui n’est pas dans le triangle

  - oreille droite d’un bretzel :: partie de la boucle droite du
    bretzel qui n’est pas dans le triangle

  - faire un bretzel droit supérieur :: faire une boucle, au-dessus
    vers la droite, puis faire tourner la boucle d’un demi-tour :

    + autour de l’axe qui relie les deux bords
    + en commençant à tourner vers le haut et l’avant
    + de façon à englober le croisement de la boucle

    Les parties libres sont orientée comme suit :

    + partie de travail vers l’avant droite
    + partie statique vers l’avant gauche

  - faire un bretzel droit inférieur :: faire une boucle, au-dessus

    + autour de l’axe qui relie les deux bords
    + en commençant à tourner vers le bas et l’avant
    + de façon à englober le croisement de la boucle

    Les parties libres sont orientée comme suit :

    + partie de travail vers l’avant droite
    + partie statique vers l’avant gauche

  - faire un bretzel gauche supérieur :: faire une boucle, au-dessus

    + autour de l’axe qui relie les deux bords

    extérieur du huitain

    croisements de la voûte de *B* avec les parties libres créent deux
    nouvelles boucles entrecroisées : *G* à gauche et *D* à droite,
    telles que :

    + l’intersection *I* de *G* et *D* est identique à *L*
    + la partie de corde *P* qui relie les croisements de *G* et de
      *D* correspond plus ou moins à la voûte de *B*
    + le bord gauche de *G* correspond plus ou moins au bord gauche de *B*
    + le bord droit de *G* contient le bord droit de *L*
    + le bord gauche de *D* contient le bord gauche de *L*
    + le bord droit de *D* correspond plus ou moins au bord droit de *B*

      perpendiculaire à l’axe qui relie le croisement à la voûte

    plus grande que la lentille.

  - intersection d’un entrecroisement :: intersection des boucles de
    l’entrecroisement

  - socle d’un entrecroisement :: partie de corde qui relie les boucles
    de l’entrecroisement

  - faire un entrecroisement droit supérieur :: faire un huitain
    supérieur droit, puis faire tourner la boucle du huitain d’un
    demi-tour :

    + de façon à englober la lentille et le croisement extérieur du
      huitain

  - faire un entrecroisement droit inférieur :: faire un huitain
    supérieur droit, puis faire tourner la boucle du huitain d’un
    demi-tour :

    + autour de l’axe qui relie les deux bords
    + en commençant à tourner vers le bas et l’avant
    + de façon à englober la lentille et le croisement extérieur du

  - faire un entrecroisement gauche supérieur :: faire un huitain
    supérieur gauche, puis faire tourner la boucle du huitain d’un
    demi-tour :

    + de façon à englober la lentille et le croisement extérieur du
      huitain

* Glossaire anglais

** Corde

  - working end / tail :: partie de la corde qui se déplace pour
    réaliser le noeud

  - standing part / lead rope :: partie statique de la corde

** Formes

  - bight :: anse, doublure

  - loop :: boucle

  - ampersand :: esperluette

** Catégories de noeuds

  - knot :: noeud

  - loop :: noeud en boucle

  - bend :: noeud d’ajut

    Attache de deux extrémités, ou plus précisément de deux terminus.

    Les deux terminus peuvent appartenir à une même corde ou à deux
    cordes distinctes.

  - stopper :: noeud d’arrêt

  - coil :: bobine

  - hitch :: accroche, noeud d’attache à un objet, souvent une barre fixe

  - lashing :: noeud d’attache de deux ou plusieurs objets

  - leverage :: levier

  - braid :: tresse

  - mat :: natte

** Propriétés des noeuds

  - loose :: non serré

  - quick release :: dénouage rapide

  - noose :: boucle réglable (noeud coulant)

  - adjustable grip :: boucle ajustable hors tension, mais fixe sous
    tension

  - gripping hitch :: hitch anti-glissement, pour traction en partie
    ou totalement parallèle au poste

** Techniques

  - hand wrap :: enroulement autour de la main

  - rewoven :: pour un noeud d’ajut, réaliser la seconde partie du
    noeud en utilisant la seconde corde pour retracer à l’envers le
    parcours de la première corde

  - running *N* :: nouer un noeud de type *N* pour former une petite
    boucle *B*, puis dessiner une grande boucle avec une des parties
    libres et la passer dans *B*

    Cette technique permet de réaliser une grande boucle ajustable à
    partir d’une autre boucle.

** Divers

  - post :: objet auquel on attache une corde
