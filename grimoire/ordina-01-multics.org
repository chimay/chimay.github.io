** Menu du BIOS

Pour accéder au menu du BIOS, c’est souvent l’une de ces touches :

- F2
- del
- F1
- F9
- F10
- F11
- F12

** Menu de boot

Pour accéder au menu de boot au démarrage, et pouvoir choisir de
démarrer sur une clé usb, un cd/dvd ou le disque interne, c’est
souvent l’une de ces touches :

- Escape
- F7

** Shell EFI

Pour accéder au shell EFI, souvent une de ces touches :

- F6
- F11
- F12

**** Vérifier le réseau

#+begin_src shell
ip addr show
#+end_src

Si pas de réseau, remplacer /enpMsN/ par la carte réseau listée par la
commande précédente

#+begin_src shell
dhcpcd -n enp2s0
#+end_src


**** Table des partitions

Schéma de la table des partitions :

  - ESP : EFI System Partition, de type ef00
  - Boot Partition, de type linux, code 8300
  - Root Partition, de type linux, code 8300

Création de la table des partitions avec /gdisk/

#+begin_src shell
gdisk /dev/sda
#+end_src

Dans /gdisk/ :

#+begin_src shell
Command : p
Command : n
Partition number [1] : <Enter>
First sector [2048] : <Enter> (minimum 2048)
Last sector : +500M (ou +1G)
Hex code or GUID [8300] : ef00
Command : p
Command : n
Partition number [2] : <Enter>
First sector [...] : <Enter>
Last sector : +500M (ou +1G)
Hex code or GUID [8300] : <Enter>
Command : p
Command : n
Partition number [3] : <Enter>
First sector [...] : <Enter>
Last sector : <Enter> (reste du disque)
Hex code or GUID [8300] : <Enter>
Command : p
Command : w
#+end_src

Vérification :

#+begin_src shell
gdisk -l /dev/sda
#+end_src


**** Formattage des partitions

Schéma du formattage :

  - ESP : EFI System Partition

    + Type : FAT32

    + Label : EFI

    + Remarque : le FAT32 ne supporte pas complètement les minuscules
      dans le label

  - Boot Partition

    + Type : Ext4

    + Label : fs_boot

  - Root Partition

    + Type : Ext4

    + Label : fs_root

Partition ESP :

#+begin_src shell
mkfs.fat -F 32 -n EFI /dev/sda1
#+end_src

Pour que refind puisse trouver le noyau linux, on formate la partition
/boot avec l’adressage en 48 bits :

#+begin_src shell
mkfs.ext4 -L fs_boot -O '^64bit' /dev/sda2
#+end_src

L’adressage en 64 bits est alors désactivé. Si malgré tout, on
souhaite formatter la partition /boot avec adressage 64 bits activé
(déconseillé) :

#+begin_src shell
mkfs.ext4 -L fs_boot /dev/sda2
#+end_src

Partition racine / :

#+begin_src shell
mkfs.ext4 -L fs_root /dev/sda3
#+end_src


**** Montage des partitions

Schéma du montage :

  - ESP : /dev/sda1 : /mnt/boot/efi

  - /dev/sda2 : /mnt/boot

  - /dev/sda3 : /mnt

#+begin_src shell
mount /dev/sda3 /mnt
mkdir /mnt/boot
mount /dev/sda2 /mnt/boot
mkdir /mnt/boot/efi
mount /dev/sda1 /mnt/boot/efi
#+end_src

**** Post-install configuration

On génère le fichier contenant la table des systèmes de fichier :

#+begin_src shell
genfstab -L /mnt >> /mnt/etc/fstab
#+end_src

On passe en change root :

#+begin_src shell
arch-chroot /mnt /bin/bash
#+end_src

Nom de l’ordinateur :

#+begin_src shell
echo nom_ordi > /etc/hostname
#+end_src

Fuseau horaire :

#+begin_src shell
ln -sf /usr/share/zoneinfo/Europe/Paris /etc/localtime
#+end_src

Localisation. Éditer :

#+begin_src shell
/etc/locale.gen
#+end_src

Décommenter la ou les lignes que vous souhaiter intégrer à votre
localisation et encodage (souvent UTF-8, voire ISO-... en complément).
Ensuite :

#+begin_src shell
locale-gen
#+end_src

Pour la localisation par défaut, éditer :

#+begin_src shell
/etc/locale.conf
#+end_src

et assigner à la variable LANG la localisation et encodage par défaut,
par exemple :

#+begin_src shell
LANG="fr_FR.UTF-8"
LC_COLLATE=C
#+end_src

Pour configurer le clavier en mode console virtuelle, on édite :

#+begin_src shell
/etc/vconsole.conf
#+end_src

et on ajoute :

#+begin_src shell
KEYMAP=fr-latin9
FONT=lat9w-16
#+end_src

Génération de l’image init cpio, qui contient les drivers nécessaires
au boot :

#+begin_src shell
mkinitcpio -p linux
#+end_src

Même chose avec linux-lts & linux-zen si vous les avez installés.

Mot de passe root :

#+begin_src shell
passwd
#+end_src


**** Installation de refind

Installation du paquet :

#+begin_src shell
pacman -S refind
#+end_src

Installation sur la partition ESP :

#+begin_src shell
refind-install
#+end_src

La commande ci-dessous précise où se situe la partition efi :

#+begin_src shell
refind-install --usedefault /dev/sda1
#+end_src

La commande ci-dessous installe tous les drivers pour systèmes de
fichiers dans la partition efi. Peut être utile dans certains cas, comme
un démarrage depuis un périphérique usb, mais peut aussi causer des
problèmes en cas de démarrage sur disque interne :

#+begin_src shell
refind-install --alldrivers
#+end_src

Ensuite, on génère /refind_linux.conf/ :

#+begin_src shell
mkrlconf
#+end_src

Voir aussi la page :

https://wiki.archlinux.org/index.php/REFInd

notamment pour les hooks lors de l’update du paquet.


**** Démontage des partitions

#+end_src

#+begin_src shell

#+end_src

ou :

#+begin_src shell

#+end_src
