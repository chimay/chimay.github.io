* Notation

Afin d’alléger les notations, nous notons :

$$ \lim_{h \to 0} = \lim_{ \substack{ h \to 0 \\ h \ne 0 } } $$

$$ \lim_{b \to a} = \lim_{ \substack{ b \to a \\ b \ne a } } $$

tout au long de ce chapitre.

* Dérivées ordinaires

Soit une fonction $f : \corps \mapsto \corps$. La dérivée ordinaire
est alors simplement :

$$\partial f(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

** Notation

On note aussi :

$$\OD{f}{x}(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

On a alors :

$$f(a + h) = f(a) + \OD{f}{x}(a) \cdot h + E(h)$$

et :

$$\lim_{h \to 0} \frac{E(h)}{h} = \lim_{h \to 0} \left[ \frac{f(a + h) - f(a)}{h} - \OD{f}{x}(a) \right] = 0$$

** Définition équivalente

Si on pose $b = a + h$, on voit que $h = b - a$ et que la
convergence $h \to 0$ est équivalente à $b \to a$. On a donc :

$$\OD{f}{x}(a) = \lim_{b \to a} \frac{f(b) - f(a)}{b - a}$$

** Dérivée seconde ordinaire

Dans le cas $\Omega$ est de dimension 1, on peut l'assimiler à $\corps$,
on a alors une fonction $f : \corps \mapsto \corps$ possédant une seule
dérivée seconde, que l'on note :

$$\OOD{f}{t}(a) = \partial^2 f(a)$$
