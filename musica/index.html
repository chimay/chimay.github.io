1.1. Introduction

Les fichiers présentés dans ce chapitre sont des modèles d’orchestres disponibles en format lilypond. Pour une navigation plus rapide, le pliage de texte (folding) des éditeurs vim / neovim est inclus.

1.2. Méta-orchestre

Le fichier le plus important est le méta-orchestre, qui contient de nombreux instruments et groupes d’instruments. Vous pouvez l’utiliser pour construire un orchestre à la carte, en décommentant ou commentant les instruments et les groupes dans la section "book" à la fin du fichier, respectivement pour les activer ou les désactiver.

Le fichier neutre est pareil, mais tous les intruments et groupes sont commentés, en attente d’être choisis.

Les thèmes mélodiques peuvent s’insérer dans les définitions des variables mélodiques commençant par "voice". Par exemple, la variable "voicePianoRight" contient la mélodie de la main droite au piano.

Dans le cas d’un groupe d’instrument qui recouvre une large étendue de notes, les voix du groupe sont nommées depuis soprano (la plus aiguë) jusqu’à basse (la plus grave). Par exemple, pour un quintette à cordes, ce sera :

  • voiceStringQuintetSoprano
  • voiceStringQuintetMezzo
  • voiceStringQuintetAlto
  • voiceStringQuintetTenor
  • voiceStringQuintetBass

Ces instruments et groupes sont disposés le plus possible avec deux mélodies par portée, pour obtenir une partition plus compacte.

Les instruments individuels tirent leur nom du fait qu’on peut activer un seul type d’instrument à la fois. Certains d’entre-eux sont en fait disponibles par paires : flûtes, hautbois, trompettes, trombones, violons, violoncelles, etc. Chaque paire d’instruments est alors representés comme deux mélodies disposées sur une même portée. Si vous avez uniquement besoin d’un seul de ces instruments, il est possible de ne remplir que la variable mélodique du premier, par exmple "voiceFluteOne" pour la première flûte. Pour retrouver la représentation habituelle des notes sur une portée à une voix, on peut alors insérer "\oneVoice" au début de la variable.
