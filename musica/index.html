1.1. Introduction

Les fichiers présentés dans ce chapitre sont des modèles d’orchestres disponibles en format lilypond. Pour une navigation plus rapide, le pliage de texte (folding) des éditeurs vim / neovim est inclus.

1.2. Fichiers principaux

Le fichier le plus important est le château, méta-orchestre qui contient de nombreux instruments et groupes d’instruments. Vous pouvez l’utiliser pour construire un orchestre à la carte en décommentant ou commentant les instruments et les groupes, respectivement pour les activer ou les désactiver.

Les autres modèles d’orchestres sont dérivés du fichier château :

  • quatuor vocal
  • septuor vocal
  • ensemble d’instruments à vent
  • ensemble d’instruments à cordes
  • septuor à cordes
  • boudoir : flûtes, quatuor à cordes, clavier
  • ensemble de musique de chambre
  • claviers : orgue, organista, clavecin, piano, percussions
  • jazz

Quelques exemples de partitions (au format pdf) générées avec ces fichiers :

  • château
  • quatuor vocal
  • septuor vocal
  • ensemble d’instruments à vent
  • ensemble d’instruments à cordes
  • septuor à cordes
  • boudoir

  • jazz
