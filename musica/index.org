** Introduction

Les fichiers présentés dans ce chapitre sont des modèles d’orchestres
disponibles en format [[https://lilypond.org/][lilypond]]. Pour une
navigation plus rapide, le pliage de texte (folding) des éditeurs
[[https://www.vim.org/][vim]] / [[https://neovim.io/][neovim]] est inclus.

** Fichiers principaux

Le fichier le plus important est le
[[file:lilypond/template/chateau.ly][château]], méta-orchestre qui
contient de nombreux instruments et groupes d’instruments. Vous pouvez
l’utiliser pour construire un orchestre à la carte en décommentant
ou commentant les instruments et les groupes dans la section "book"
à la fin du fichier, respectivement pour les activer ou les désactiver.

Les autres modèles d’orchestres sont dérivés du fichier
château. Voici quelques exemples de partitions (au format pdf)
