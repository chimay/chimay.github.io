* Modèles lilypond

#+TOC: headlines 1 local

** Introduction

Les fichiers présentés dans ce chapitre sont des modèles d’orchestres
disponibles en format [[https://lilypond.org/][lilypond]]. Pour une
navigation plus rapide, le pliage de texte (folding) des éditeurs
[[https://www.vim.org/][vim]] / [[https://neovim.io/][neovim]] est inclus.

** Fichiers principaux

Le fichier le plus important est le
[[file:template/chateau.org][château]], méta-orchestre qui contient de
nombreux instruments et groupes d’instruments. Vous pouvez l’utiliser
pour construire un orchestre à la carte en décommentant ou commentant
les instruments et les groupes, respectivement pour les activer ou
les désactiver.

Les autres modèles d’orchestres sont principalement des sous-ensembles
du fichier château. En voici des aperçus :

- [[file:template/septuor-chant.org][septuor vocal]]
- [[file:template/septuor-cordes.org][septuor à cordes]]
- [[file:template/boudoir.org][boudoir]] : flûtes, quatuor à cordes, clavier
- [[file:template/clavier.org][clavier]] : orgue, organista, clavecin, piano, percussions
- [[file:template/jazz.org][jazz]] : bois, cuivres, orgue, piano, guitares, percussions

Vous pouvez trouver la version texte de ces fichiers modèles ici :

- [[file:template/lilypond/chateau.ly][château]]
- [[file:template/lilypond/septuor-chant.ly][septuor vocal]]
- [[file:template/lilypond/septuor-cordes.ly][septuor à cordes]]
- [[file:template/lilypond/boudoir.ly][boudoir]]
- [[file:template/lilypond/clavier.ly][clavier]]
- [[file:template/lilypond/jazz.ly][jazz]]
