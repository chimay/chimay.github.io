** Introduction

Les fichiers présentés dans ce chapitre sont des modèles d’orchestres
disponibles en format [[https://lilypond.org/][lilypond]]. Pour une
navigation plus rapide, le pliage de texte (folding) des éditeurs
[[https://www.vim.org/][vim]] / [[https://neovim.io/][neovim]] est inclus.

** Fichiers principaux

Le fichier le plus important est le
[[file:lilypond/template/meta.ly][méta]] orchestre, qui contient
de nombreux instruments et groupes d’instruments. Vous pouvez
l’utiliser pour construire un orchestre à la carte en décommentant
ou commentant les instruments et les groupes dans la section "book" à
la fin du fichier, respectivement pour les activer ou les désactiver.
Les thèmes mélodiques peuvent s’insérer dans les définitions des
variables "voice". Par exemple, la variable "voicePianoRight" contient
la mélodie de la main droite au piano.
