* Système nerveux


** Sous-systèmes

  - Central : cerveau et moelle épinière

  - Périphérique : le reste (nerfs, etc)


** Conscient & Autonome

  - Volontaire

    + Activité soumise à la volonté

  - Autonome

    + Sympathique (Ortho-sympathique)

      * Consommation d’énergie

      * Production d’acidité

      * Catabolie

    + Para-sympathique

      * Régénération

      * Construction

      * Anabolie

      * Élimination

    + Entérique

      * Intestins


** Respiration

    + Inspiration

      * Système nerveux sympathique

    + Expiration

      * Système nerveux para-sympathique


** Nerf vague

  - Communication entre système nerveux central et intestins

  - Active le péristaltisme intestinal

  - Gargariser stimule le nerf vague


* Hormones

  - Dopamine : stimule l’organisme

  - Sérotonine : permet à l’organisme de se régénérer

  - Cholestérol : permet la fabrication de plusieurs hormones

    + Une hormone qui active le système para-sympathique

    + Une hormone qui permet au coeur de transformer les toxines
      produites en un produit dont il peut se nourrir

  - Cortisol

    + Régule l’inflammation

    + Remise en circulation des réserves

    + Contracte les vaisseaux sanguins

  - Adrénaline

    + Contracte les vaisseaux sanguins

  - PTH

    + Remet le calcium stocké en circulation


* Système lymphatique

  - Le mouvement permet de faire circuler le système lymphatique

    + Voir [[file:bienetre-02-enpratique.org::#heading:exercicePhysique][exercice physique]]


* Élimination

  - Émonctoires : voies éliminatoires

    + Poumons

      * Via respiration

      * Saturation --> rhume

    + Peau

      * Saturation --> boutons

    + Reins

    + Intestins

  - Les muqueuses servent de filtre

    + Poumons

    + Intestins

  - On élimine plus le soir, la nuit et le matin


* Métabolisme

  - Concentration en sels minéraux

    + Inférieure à l’isotonie

      * Ralentissement du métabolisme

      * Frilosité

      * Humeur fluctuante

      * Conseils

	- Consommer des aliments contenant moins d’eau

	- Consommer des aliments contenant des sels minéraux

	- Consommes des aliments contenant des graisses

    + Supérieure à l’isotonie

      * Accélération du métabolisme

      * Transpiration abondante

      * Conseils

	- Boire

	- Consommer des fruits juteux


* Immunité

  - Composantes

    + Au niveau des muqueuses : bactéries, champignons

    + Humorale : sang, lymphe

    + Cellulaire


* Médecine traditionnelle chinoise


** Éléments

  - Organes

    + Eau : Reins, Vessie

    + Bois : Foie, Vésicule biliaire

    + Feu : Coeur, Intestin grêle

    + Terre : Rate, Estomac

    + Métal : Poumons, Gros intestin

  - Cycles

    + Cycle nourissier

      * L’eau nourrit le bois

      * Le bois nourrit le feu

      * Le feu nourrit la terre

      * La terre nourrit le métal

      * Le métal nourrit l’eau

    + Cycle de contrôle

      * L’eau contrôle le feu

      * Le feu contrôle le métal

      * Le métal contrôle le bois

      * La bois contrôle la terre

      * La terre contrôle l’eau

  - Mouvements du corps

    + Eau : Être debout

    + Bois : Marcher

    + Feu : Courir

    + Terre : Être assis

    + Métal : Être couché

  - Saveurs

    + Eau : Salé

    + Bois : Acide

    + Feu : Amer

    + Terre : Doux, Sucré

    + Métal : Piquant

  - Émotions

    + Reins : Peur

    + Foie : Colère

    + Coeur : Joie

    + Poumons : Tristesse

    + Rate : Réflexion

  - Manifestations

    + Reins : Cheveux

    + Foie : Ongles

    + Rate : Lèvres

  - Ouvertures

    + Reins : Oreilles

    + Foie : Yeux

    + Coeur : Pointe de la langue

    + Rate : Bouche
